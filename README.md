# Quiz App 🧠

A full-stack online quiz application built with Python & Flask.

## 🔗 Live Demo
👉 [Click here to open the app](https://web-production-de7be.up.railway.app)

## Features
- Multi-subject quizzes (Python, Java, Math, GK, Web Dev)
- Timed quiz with countdown
- Easy / Medium / Hard difficulty levels
- User login & registration
- Global leaderboard
- Admin panel to add/delete questions

## Tech Stack
- Python, Flask, SQLite
- HTML, CSS, JavaScript
